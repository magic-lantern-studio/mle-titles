This directory contains an example title for the Qt target. The title is
"imagetest" for the Magic Lantern SDK targeting Qt applications. Essentially
the imagetest title displays the image passed in on the command line. It does
not utilize the Digital Workprint or Playprint technologies.

1. Build the title with the generated source. The Qt project can be found in the
   $MLE_HOME/Titles/imagetest/qt/imagetest/imagetest.pro file.
